Recursive Feature Elimination by Sensitivity Testing

There is great interest in methods to improve human insight into trained non-linear models such as support vector machines (SVMs), deep neural networks, and large random forests. We show theoretically and empirically the benefit of a novel version of recursive feature elimination (RFE) called recursive feature elimination by sensitivity testing (RFEST).

The goal of using RFEST is to discover relevant features in your dataset such that their combination is higher in predictive power, quality, and better interpretability. My empirical results use a real-world cancer genomics problem (demographic and genomic data) and on synthetic data where ground truth is known. Due to privacy constraints, I show how the algorithm can be used on a synthetic dataset (refer to github repository found below).

  • Technologies: R, Python

  • PI: Dr. David Page

Lane Detection in Adverse Conditions

Lane detection and tracking has become a prominent area as the push for autonomous vehicles surges. Reasons for doing so include automating and enhancing the driving experience. Current state-of-the-art involve using a combination of information (i.e. LiDAR, GPS, vehicle sensor information). This project focused on solely RGB input images. The primary goal was to show how we can create a robust solution for illumination problems (e.g. shadows). In turn, we reimplemented an approach that used an enhanced-gradient conversion and adaptive Canny edge detection in their lane detection algorithm.

  • Technologies: Python

Evaluation of Gene-Environment Interaction for Ovarian Cancer

This project analyzed the relationship between a cancer susceptibility gene, referred to as BRCA1/2, and external risk factors (parity and oral contraceptive use in particular) on the incidence of ovarian cancer in women.

Our results showed that there was not only a significant interaction between gene and environmental factors, but that oral contraceptive use and parity were not effective in preventing ovarian cancer in women with the BRCA1/2 mutation relative to those without the gene mutation.

  • Technologies: R

  • PI: Dr. Eric T. Tchetgen
  • Presentation: First Annual Pipelines into Biostatistics Symposium
